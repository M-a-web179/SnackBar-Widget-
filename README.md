# SnackBar Undo Delete Demo

A tiny Flutter todo list app that demonstrates how to use `SnackBar` for an **“undo delete”** interaction in a realistic scenario.

## Overview

- The home screen shows a list of simple tasks.
- Tapping the trash icon deletes a task and shows a `SnackBar` with an **UNDO** action.
- Pressing UNDO restores the deleted task to its original position in the list.

This mirrors common behavior in apps like email or notes, where destructive actions can be reversed briefly.
